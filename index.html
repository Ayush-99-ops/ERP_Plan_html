Month 3 — Customization & Reports
WK 09–10
Custom App
DVS Custom Frappe App + Workflows
Jul 27–Aug 9
▾

Key Activities

  • Create and maintain a dedicated DVS custom Frappe app
  • Add approved custom fields to existing doctypes
  • Configure workflows and approval chains
  • Customize quotation format and document templates
  • Customize project/job stages based on DVS operations

What "DVS Custom App" Means

  • A Frappe app — Python/JS, not Flutter or mobile
  • Installed on the same Docker server as ERPNext
  • Contains only DVS-specific fields, workflows, print formats
  • Does NOT modify ERPNext core — safe to update later

Month 5 — Production Deployment, Training & Go-Live
WK 17–18
Production
Production AWS Setup + Final Deployment
Sep 21–Oct 4
▾

Key Activities

  • Prepare production AWS environment (separate from staging)
  • Configure domain/subdomain, SSL, backups, and CloudWatch monitoring
  • Deploy ERPNext production environment with all custom apps
  • Import approved final data (employees, clients, projects)
  • Configure production users and permissions

Important Distinctions

  • Staging = where you tested. Production = where they work
  • Never develop or test directly on production
  • Final data import is a one-time careful operation
  • Buy reserved instance now — saves 30-35% monthly
